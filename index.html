Portfolio: Majken Overgaard

The task in this project was to construct a personalised Portfolio for the arts-curator Majken Overgaard. She wished to have a unique website that doesn't follow the ordinary structure seen on the internet. Specifically, Majken was against the structure observed in Wordpress or Squarespace websites. Majken also had a wish to have a retro vibe, that was reminiscing of the early days of the internet.

This website was made to resemble an old windows desktop. The icons work as the navigation system, each icon will open what looks like an internet-window, where inside is the listed projects showcased. Each window can be closed by ether by pressing the “home-button” in the button right corner, pressing the “exit-button” inside the windows top right corner or opening another window. In the button right corner we have a pop-up menu, that when hovered over will show the up and coming events. The background was borrowed with permission from an art piece form Line Finderup Jensen, called IRL-URL.

This project was constructed with Visual Studio Code. Only the code languages used are HTML, CSS and Javascript. We made all the icons ourselves.

This project was constructed: October 2023
